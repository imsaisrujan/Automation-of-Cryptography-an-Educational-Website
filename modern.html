Modern Techniques

Modern cryptography is the cornerstone of computer and communications security. Its foundation is based on various concepts of mathematics such as number theory, computational-complexity theory, and probability theory.

It relies on publicly known mathematical algorithms for coding the information. Secrecy is obtained through a secrete key which is used as the seed for the algorithms. The computational difficulty of algorithms, absence of secret key, etc., make it impossible for an attacker to obtain the original information even if he knows the algorithm used for coding.

Advanced Encryption Standard (AES)

The AES Encryption algorithm (also known as the Rijndael algorithm) is a symmetric block cipher algorithm with a block/chunk size of 128 bits. It converts these individual blocks using keys of 128, 192, and 256 bits. Once it encrypts these blocks, it joins them together to form the ciphertext.

Data Encryption Standard (DES)

Data encryption standard (DES) is a symmetric block cipher and encrypts data in blocks of size of 64 bits each, which means 64 bits of plain text go as the input to DES, which produces 64 bits of ciphertext. The same algorithm and key are used for encryption and decryption, with minor differences. The key length is 56 bits.

Elliptic Curve Cryptography (ECC)

Elliptic Curve Cryptography (ECC) is a key-based technique for encrypting data. ECC focuses on pairs of public and private keys for decryption and encryption of web traffic. It generates security between key pairs for public key encryption by using the mathematics of elliptic curves.

Rivest-Shamir-Adleman(RSA)

RSA is an asymmetric encryption algorithm that uses a pair of keys to encrypt and decrypt data. The key pair consists of a public key and a private key, with the public key being accessible to anyone and the private key only known to the creator. RSA encryption uses opposite keys to encrypt and decrypt data.
